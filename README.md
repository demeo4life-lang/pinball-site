# Pinball Trading Co Website

This is a simple static website for **Pinball Trading Co**, a business specializing in the sale, rental, and purchase of pinball and arcade machines across Sydney, Central Coast, Newcastle, and Hunter Valley.

## Features

- Pinball machines for sale with warranty and free delivery
- Pinball machines for hire for events and venues
- We buy machines in any condition
- Free claw and chocolate machines for qualifying venues
- Contact section with direct phone and email info

## Tech Stack

- HTML5
- CSS3
- Hosted via GitHub Pages
